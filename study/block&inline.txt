1. 블록요소
    1) 종류 : DIV, H1, P 등
    2) 특징 : 사용 가능한 최대 가로 너비를 사용
    3) 크기 : 지정 가능
    4) 시작 : width : 100% / height: 0%
    5) 축적 : 수직
    6) margin, padding : 상하좌우 가능
    7) 주로 LAYOUT

2. 인라인요소
    1) 종류 : SPAN, IMG 등
    2) 특징 : 필요한 만큼의 너비를 사용
    3) 크기 : 지정 불가
    4) 시작 : width : 0 / height: 0
    5) 축적 : 수평
    6) margin, padding : 좌우만 가능
    7) 주로 TEXT

※ display라는 속성의 영향을 받는다 (inline, block)
